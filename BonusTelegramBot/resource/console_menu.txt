**Главное меню консоли бота**
Список доступных команд:
    /getprofile <id> - Показать профиль в боте по ID (ID:*, чтобы показать все)
    /block <id> - Заблокировать профиль в боте по ID
    /unblock <id> - Разблокировать профиль в боте по ID
    /setbalance <id> <value> - Установить баланс профиля в боте по ID
    /getbills - Показать все выставленные счета в боте
    /sendmsg <id> <msg> - Отправить сообщение профилю в боте по ID (ID:*, чтобы отправить всем)
    /usage - Мониторинг ресурсов программы
    /stop - Остановить бота и программу
